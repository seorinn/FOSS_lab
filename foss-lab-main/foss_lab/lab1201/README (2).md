# 1201 노트필기 - 오전
```
KPI: key Performance Indicator 
BSC: Balanced Score Card 
KPI와 전략체계와 BSC가 왜 연결되어 있는가 하면 각각의 프로젝트에 적절한 KPI가 있어야 하기 때문
전략 목표들이(KPI가) 어떻게 수립되어야 하는가: 
Balance가 중요 - 재무적/비재무적 KPI가 balance. 장기적/단기적인 것도 balance.
BSC는 실제로 구현 힘듦 - KPI의 기본적인 설정이 쉽지 않음. 도전적인 KPI
왜 소프트웨어를 product가 아닌 service로 만들려고 할까: 
Maintenance risk x
대표적인 대안 -> free and open source software
```
